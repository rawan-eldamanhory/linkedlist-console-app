# C# LinkedList Implementation

This project demonstrates the usage of the `LinkedList<T>` collection in C# through a simple console application.

## Concepts Covered
- Add elements at the beginning and end (`AddFirst`, `AddLast`)
- Remove elements (`Remove`, `RemoveFirst`, `RemoveLast`)
- Insert elements relative to a node (`AddAfter`, `AddBefore`)
- Check element existence (`Contains`)
- Iterate and display elements
- Clearing the linked list (`Clear`)

## Technologies
- C#
- Console Application
- Visual Studio

## Purpose
This project is intended for learning and practicing C# collection fundamentals, focusing on the LinkedList data structure.
